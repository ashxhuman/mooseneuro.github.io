15/**
16 * This class handles a single gate on an HHChannel. It is equivalent to the
17 * m and h terms on the Hodgkin-Huxley Na channel, or the n term on the
18 * K channel. It stores the
19 * voltage-dependence (sometimes concentration-dependence) of the
20 * gating variables for opening the channel. It does so in a tabular form
21 * which can be directly filled using experimental data points.
22 * It also provides a set of
23 * utility functions for defining the gate in functional forms, and
24 * accessing those original functional forms.
25 * The HHGate is
26 * accessed as a FieldElement, which means that it is available as a
27 * pointer on the HHChannel. HHGates are typically shared. This means that
28 * when you make a copy or a vector of an HHChannel, there is only a single
29 * HHGate created, and its pointer is used by all the copies.
30 * The lookup functions are thread-safe.
31 * Field assignment to the HHGate should be possible only from the
32 * original HHChannel, but all the others do have read permission.
33 */
34
