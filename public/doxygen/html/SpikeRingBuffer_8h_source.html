12/**
13 * This ring buffer handles incoming spikes. It spans an interval equal to
14 * the longest arrival delay. When a spike event is notified it puts it into
15 * the slot defined by the arrival time. It just adds the weight onto the
16 * existing value in this slot, assuming a linear summation of the
17 * weights of coincident inputs.
18 */
