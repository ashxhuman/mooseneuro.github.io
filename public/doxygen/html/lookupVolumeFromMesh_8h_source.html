32/**
33 * Generates conversion factor for rates from concentration to mol# units.
34 * Assumes that all reactant pools (substrate and product) are within the
35 * same mesh entry and therefore have the same volume.
36 * The outcome of this calculation is:
37 * kf = Kf * convertConcToNumRate.
38 * The Eref is an Enz or Reac.
39 * The SrcFinfo is a message to the pools.
40 * The meshIndex specifies the mesh voxel to use.
41 * The scale term is a conversion from the conc units to SI:
42 * The scale term is 1 if conc units are in SI, which is
43 * equal to moles per cubic metre, which is equal to millimolar.
44 * The scale term is 1e-3 for micromolar, uM.
45 * The doPartialConversion flag tells the function that there are other
46 * substrates not in the 'pools' list, and so it should compute the
47 * conversion for all pools, not n-1. This flag defaults to 0.
48 */
49double convertConcToNumRateUsingMesh( const Eref& e, const SrcFinfo* pools,
50 bool doPartialConversion );
51
52/**
53 * Generates conversion factor for rates from concentration to mol# units.
54 * This variant already knows the volume, but has to figure out # of
55 * reactants.
56 * The scale term is a conversion from the conc units to SI:
57 * The scale term is 1 if conc units are in SI, which is
58 * equal to moles per cubic metre, which is equal to millimolar.
59 * The scale term is 1e-3 for micromolar, uM.
60 * The doPartialConversion flag tells the function that there are other
61 * substrates not in the 'pools' list, and so it should compute the
62 * conversion for all pools, not n-1. This flag defaults to 0.
63 */
64double convertConcToNumRateUsingVol( const Eref& e, const SrcFinfo* pools,
65 double volume, double scale, bool doPartialConversion );
66
67/**
68 * Generates conversion factor for rates from concentration to mol# units.
69 * This variant is used when the reactants are in different compartments
70 * or mesh entries, and may therefore have different volumes.
71 * We already know the reactants and their affiliations.
72 * We use the first vol, v1, as the reference.
73 * The outcome of this calculation is:
74 * kf = Kf * convertConcToNumRate.
75 * The scale term is a conversion from the conc units to SI:
76 * The scale term is 1 if conc units are in SI, which is
77 * equal to moles per cubic metre, which is equal to millimolar.
78 * The scale term is 1e-3 for micromolar, uM.
79 */
80double convertConcToNumRateInTwoCompts( double v1, unsigned int n1,
81 double v2, unsigned int n2, double scale );
