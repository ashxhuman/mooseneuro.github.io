14/**
15 * This is the base class for CaConc. It handles calcium current influx,
16 * conversion to calcium concentration, buffering, and pumps.
17 * It obeys the equation:
18 *
19 * dC/dt = B*Ik - C/tau
20 * where Ca = Ca_base + C.
21 *
22 * From the GENESIS notes:
23 * In SI units, where concentration is moles/m^3
24 * (milli-moles/liter) and current is in amperes, theory gives B
25 * = 5.2e-6/(shell volume). In practice, B is a parameter to be
26 * fitted or estimated from experiment, as buffering, non-uniform
27 * distribution of Ca, etc., will modify this value. If thick =
28 * 0, the readcell routine calculates B by dividing the "density"
29 * parameter in the cell parameter file by the volume of the
30 * compartment. Otherwise, it scales as a true shell, with the
31 * volume of a shell having thickness thick. A negative value of
32 * the "density" parameter may be used to indicate that it should
33 * be taken as an absolute value of B, without scaling.
34 *
35 * This is a base class to provide the interface functions.
36 */
37
