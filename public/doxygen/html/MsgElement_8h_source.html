15/**
16 * This is a specialized Element class for managing Msg data.
17 * Basically it just gives a wrapper for the vectors managed by each
18 * of the Msg classes.

22 public:

29 * name is its name
30 */

129 /**
130 * Inherited virtual.
131 * Looks up specified field data entry. On regular objects just
132 * returns the data entry specified by the rawIndex.
133 * On FieldElements like synapses, does a second lookup on the
134 * field index.
135 * Note that the index is NOT a
136 * DataIndex: it is instead the raw index of the data on the current
137 * node. Index is also NOT the character offset, but the index
138 * to the data array in whatever type the data may be.
139 *
140 * The DataIndex has to be filtered through the nodeMap to
141 * find a) if the entry is here, and b) what its raw index is.
142 *
143 * Returns 0 if either index is out of range.
144 */
145 char* data( unsigned int rawIndex,
146 unsigned int fieldIndex = 0 ) const
147 {
148 return lookupMsg_( rawIndex );
149 }
