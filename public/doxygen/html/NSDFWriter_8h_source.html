63/**
64 NSDFWriter dumps data in NSDF file format.
65
66 - As of June 2015, MOOSE uses fixed time steps for updating field
67 values. So we support only Uniform data.

69 - This class has one SrcFinfo to request data. Multiple get{Field}

75
76 */
