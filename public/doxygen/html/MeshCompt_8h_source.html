13/**
14 * The MeshCompt represents the base class for all compartments which
15 * are set up as a meshes. These include CubeMesh, CylMesh and NeuroMesh.
16 * The MeshCompt handles the stencil that is the core of all the meshes.
17 * The stencil specifies which compartments diffuse into each other, and by
18 * how much.
19 */

21{
22 public:
23 MeshCompt();
