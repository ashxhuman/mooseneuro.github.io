Detailed Description

This pure virtual base class is for solvers that want to talk to pools. The Eref specifies both the pool identity and the voxel number within the pool.

◆ getBlock()

virtual void KsolveBase::getBlock ( vector< double > &  values) const
pure virtual

Gets block of data. The first 4 entries are passed in on the 'values' vector: the start voxel, numVoxels, start pool#, numPools. These are followed by numVoxels * numPools of data values which are filled in by the function. We assert that the entire requested block is present in this KsolveBase. The block is organized as an array of arrays of voxels; values[pool#][voxel#]

Note that numVoxels and numPools are the number in the current block, not the upper limit of the block. So values.size() == 4 + numPools * numVoxels.
