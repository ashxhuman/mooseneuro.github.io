10/**
11 * this class is used to take data that has already been converted into
12 * a buffer by one or more Conv<> operations, and send it between nodes.
13 * Useful when one doesn't know the data type being transferred.
14 */

17 public:
18 /**
19 * Constructor.
20 * Here data is a pointer to the entire data block.
21 * dataSize is the size of the entire data block to be transferred,
22 * dataSize = individualDataSize * numEntries.
23 * numEntries is the # of array entries. For non-arrays it defaults
24 * to 0.
25 */
