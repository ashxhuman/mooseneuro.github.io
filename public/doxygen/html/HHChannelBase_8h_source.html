21/**
22 * The HHChannelBase is the base class for defining Hodgkin-Huxley type
23 * channels, specifically dealing with derivatives used in the HSolver.
24 * This is derived from the ChanBase rather than the ChanCommon, since
25 * the Zombie classes used in the HSolver will not use the ChanCommon
26 * fields.
27 */
