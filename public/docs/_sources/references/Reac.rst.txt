Reac
====


Class for reactions. Handles both standalone andsolved configurations

Author:   Upinder S. Bhalla, 2012, 2020 NCBS

.. describe:: Kb (type: double, class: Reac)

   Reverse rate constant, in concentration units

.. describe:: Kf (type: double, class: Reac)

   Forward rate constant, in concentration units

.. describe:: children (type: vector<Id>, class: Reac)

   vector of ObjIds listing all children of current object

.. describe:: className (type: string, class: Reac)

   Class Name of object

.. describe:: compartment (type: ObjId, class: Reac)

   ObjId of parent compartment of Reac. If the compartment isn'tavailable
   this returns the root ObjId.

.. describe:: destFields (type: vector<string>, class: Reac)

   List of all destination fields on Element, that is, fieldsthat are accessed
   as Element functions.

.. describe:: dt (type: double, class: Reac)

   Timestep used for this Element. Zero if not scheduled.

.. describe:: fieldIndex (type: unsigned int, class: Reac)

   For a FieldElement: field Index of self.For a regular Element: zero.

.. describe:: idValue (type: unsigned int, class: Reac)

   Object id of self, converted to an unsigned int.

.. describe:: index (type: unsigned int, class: Reac)

   For a FieldElement: Object index of parent.For a regular Element: Object
   index (dataId) of self.

.. describe:: me (type: ObjId, class: Reac)

   ObjId for current object

.. describe:: msgIn (type: vector<ObjId>, class: Reac)

   Messages coming in to this Element

.. describe:: msgOut (type: vector<ObjId>, class: Reac)

   Messages going out from this Element

.. describe:: name (type: string, class: Reac)

   Name of object

.. describe:: numData (type: unsigned int, class: Reac)

   # of Data entries on Element.Note that on a FieldElement this does NOT
   refer to field entries,but to the number of DataEntries on the parent
   of the FieldElement.

.. describe:: numField (type: unsigned int, class: Reac)

   For a FieldElement: number of entries of self.For a regular Element:
   One.

.. describe:: numKb (type: double, class: Reac)

   Reverse rate constant, in # units

.. describe:: numKf (type: double, class: Reac)

   Forward rate constant, in # units

.. describe:: numProducts (type: unsigned int, class: Reac)

   Number of products of reaction

.. describe:: numSubstrates (type: unsigned int, class: Reac)

   Number of substrates of reaction

.. describe:: parent (type: ObjId, class: Reac)

   Parent ObjId for current object

.. describe:: path (type: string, class: Reac)

   text path for object

.. describe:: sourceFields (type: vector<string>, class: Reac)

   List of all source fields on Element, that is fields that can act as
   message sources.

.. describe:: this (type: 7Neutral, class: Reac)

   Access function for entire object

.. describe:: tick (type: int, class: Reac)

   Clock tick for this Element for periodic execution in the main simulation
   event loop. A default is normally assigned, based on object class, but
   one can override to any value between 0 and 19. Assigning to -1 means
   that the object is disabled and will not be called during simulation
   execution The actual timestep (dt) belonging to a clock tick is defined
   by the Clock object.

.. describe:: valueFields (type: vector<string>, class: Reac)

   List of all value fields on Element.These fields are accessed through
   the assignment operations in the Python interface.These fields may also
   be accessed as functions through the set<FieldName> and get<FieldName>
   commands.
