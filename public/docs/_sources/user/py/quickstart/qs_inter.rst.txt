*********************
Interactive Tutorials
*********************

Hands-on interactive tutorials on topics in Neuroscience and System Biology are available as Jupyter notebooks in a separate github repository `moose-notebooks <https://github.com/MooseNeuro/moose-notebooks>`__.

To run these notebooks you can either install MOOSE as a python module on your own computer or install it in a `Google colab environment <https://colab.research.google.com/>`__.
