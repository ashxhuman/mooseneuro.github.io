EnzBase
=======


Abstract base class for enzymes.

Author:   Upi Bhalla


Attributes:
-----------



Value Attributes:
-----------------

.. describe:: Km (type: double, class: EnzBase)

   Michaelis-Menten constant in SI conc units (milliMolar)

.. describe:: children (type: vector<Id>, class: EnzBase)

   vector of ObjIds listing all children of current object

.. describe:: className (type: string, class: EnzBase)

   Class Name of object

.. describe:: compartment (type: ObjId, class: EnzBase)

   ObjId of parent compartment of pool. If the compartment isn'tavailable
   this returns the root ObjId.

.. describe:: destFields (type: vector<string>, class: EnzBase)

   List of all destination fields on Element, that is, fieldsthat are accessed
   as Element functions.

.. describe:: dt (type: double, class: EnzBase)

   Timestep used for this Element. Zero if not scheduled.

.. describe:: fieldIndex (type: unsigned int, class: EnzBase)

   For a FieldElement: field Index of self.For a regular Element: zero.

.. describe:: idValue (type: unsigned int, class: EnzBase)

   Object id of self, converted to an unsigned int.

.. describe:: index (type: unsigned int, class: EnzBase)

   For a FieldElement: Object index of parent.For a regular Element: Object
   index (dataId) of self.

.. describe:: kcat (type: double, class: EnzBase)

   Forward rate constant for enzyme, units 1/sec

.. describe:: me (type: ObjId, class: EnzBase)

   ObjId for current object

.. describe:: msgIn (type: vector<ObjId>, class: EnzBase)

   Messages coming in to this Element

.. describe:: msgOut (type: vector<ObjId>, class: EnzBase)

   Messages going out from this Element

.. describe:: name (type: string, class: EnzBase)

   Name of object

.. describe:: numData (type: unsigned int, class: EnzBase)

   # of Data entries on Element.Note that on a FieldElement this does NOT
   refer to field entries,but to the number of DataEntries on the parent
   of the FieldElement.

.. describe:: numField (type: unsigned int, class: EnzBase)

   For a FieldElement: number of entries of self.For a regular Element:
   One.

.. describe:: numKm (type: double, class: EnzBase)

   Michaelis-Menten constant in number units, volume dependent

.. describe:: numProducts (type: unsigned int, class: EnzBase)

   Number of products in this MM reaction. Usually 1.

.. describe:: numSubstrates (type: unsigned int, class: EnzBase)

   Number of substrates in this MM reaction. Usually 1.Does not include
   the enzyme itself

.. describe:: parent (type: ObjId, class: EnzBase)

   Parent ObjId for current object

.. describe:: path (type: string, class: EnzBase)

   text path for object

.. describe:: sourceFields (type: vector<string>, class: EnzBase)

   List of all source fields on Element, that is fields that can act as
   message sources.

.. describe:: this (type: 7Neutral, class: EnzBase)

   Access function for entire object

.. describe:: tick (type: int, class: EnzBase)

   Clock tick for this Element for periodic execution in the main simulation
   event loop. A default is normally assigned, based on object class, but
   one can override to any value between 0 and 19. Assigning to -1 means
   that the object is disabled and will not be called during simulation
   execution The actual timestep (dt) belonging to a clock tick is defined
   by the Clock object.

.. describe:: valueFields (type: vector<string>, class: EnzBase)

   List of all value fields on Element.These fields are accessed through
   the assignment operations in the Python interface.These fields may also
   be accessed as functions through the set<FieldName> and get<FieldName>
   commands.
