Value Attributes:¶

vector of ObjIds listing all children of current object

className (type: string, class: Finfo)

Class Name of object

dest (type: vector<string>, class: Finfo)

Subsidiary DestFinfos. Useful for SharedFinfos

destFields (type: vector<string>, class: Finfo)

List of all destination fields on Element, that is, fieldsthat are accessed as Element functions.

docs (type: string, class: Finfo)

Documentation for Finfo

dt (type: double, class: Finfo)

Timestep used for this Element. Zero if not scheduled.

fieldIndex (type: unsigned int, class: Finfo)

For a FieldElement: field Index of self.For a regular Element: zero.

fieldName (type: string, class: Finfo)

Name of field handled by Finfo

idValue (type: unsigned int, class: Finfo)

Object id of self, converted to an unsigned int.

index (type: unsigned int, class: Finfo)

For a FieldElement: Object index of parent.For a regular Element: Object index (dataId) of self.

me (type: ObjId, class: Finfo)

ObjId for current object

msgIn (type: vector<ObjId>, class: Finfo)

Messages coming in to this Element

msgOut (type: vector<ObjId>, class: Finfo)

Messages going out from this Element

name (type: string, class: Finfo)

Name of object

numData (type: unsigned int, class: Finfo)

# of Data entries on Element.Note that on a FieldElement this does NOT refer to field entries,but to the number of DataEntries on the parent of the FieldElement.

numField (type: unsigned int, class: Finfo)

For a FieldElement: number of entries of self.For a regular Element: One.

parent (type: ObjId, class: Finfo)

Parent ObjId for current object

path (type: string, class: Finfo)

text path for object

sourceFields (type: vector<string>, class: Finfo)

List of all source fields on Element, that is fields that can act as message sources.

src (type: vector<string>, class: Finfo)

Subsidiary SrcFinfos. Useful for SharedFinfos

this (type: 7Neutral, class: Finfo)

Access function for entire object

tick (type: int, class: Finfo)

Clock tick for this Element for periodic execution in the main simulation event loop. A default is normally assigned, based on object class, but one can override to any value between 0 and 19. Assigning to -1 means that the object is disabled and will not be called during simulation execution The actual timestep (dt) belonging to a clock tick is defined by the Clock object.

type (type: string, class: Finfo)

RTTI type info for this Finfo

valueFields (type: vector<string>, class: Finfo)

List of all value fields on Element.These fields are accessed through the assignment operations in the Python interface.These fields may also be accessed as functions through the set<FieldName> and get<FieldName> commands.
